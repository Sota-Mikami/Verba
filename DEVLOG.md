### Build Commands
```bash
# Debug (monochrome icon)
xcodebuild -project Verba.xcodeproj -scheme Verba -configuration Debug -destination "platform=macOS" build

# Release (colored icon, signed)
xcodebuild -project Verba.xcodeproj -scheme Verba -configuration Release CODE_SIGN_IDENTITY="Verba Dev" CODE_SIGNING_REQUIRED=YES CODE_SIGN_STYLE=Manual DEVELOPMENT_TEAM="" -destination "platform=macOS" build

# Copy release to /Applications
RELEASE_APP=$(find ~/Library/Developer/Xcode/DerivedData/Verba-*/Build/Products/Release -name "Verba.app" -maxdepth 1 | head -1)
cp -pR "$RELEASE_APP" /Applications/Verba.app
```
