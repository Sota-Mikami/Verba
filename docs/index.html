Raw speech in,
clean text out.

Voice input is messy — fillers, repetition, broken sentences. Verba's on-device LLM cleans it up while preserving what you meant. No cloud API key needed.

Choose from built-in prompts (general, meeting notes, email) or write your own. Supports Qwen3, Gemma 3, and SmolLM3 models via Apple MLX.

Press a key.
Start talking.

Hold fn and speak. When you release, Verba transcribes your audio, optionally cleans it up with an LLM, and pastes the result into whatever app you're using.

The floating indicator shows real-time streaming preview so you always know what's being recognized. Switch modes and prompts mid-recording, or tap Done to confirm or Cancel to discard. Works in any text field — editors, Slack, email, browser.
