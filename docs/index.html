Verba doesn't phone home. Speech recognition runs on WhisperKit via Apple Neural Engine. Text formatting uses MLX models loaded into local memory. There's no server, no analytics, no telemetry.

It works on airplane mode. That's the whole point.

Press a key.
Start talking.

Hold fn and speak. When you release, Verba transcribes your audio, optionally cleans it up with an LLM, and pastes the result into whatever app you're using.

The floating indicator shows real-time streaming preview so you always know what's being recognized. It works in any text field — editors, Slack, email, browser.

Raw speech in,
clean text out.

Voice input is messy — fillers, repetition, broken sentences. Verba's on-device LLM cleans it up while preserving what you meant. No cloud API key needed.

Choose from built-in prompts (general, meeting notes, email) or write your own. Supports Qwen3, Gemma 3, and SmolLM3 models via Apple MLX.
